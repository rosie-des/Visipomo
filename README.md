# Visipomo

## Overview

Where daily focus meets long-term success

A Pomodoro-style focus timer app built for people who want to stay productive and motivated in one place. Combines deep work sessions with vision boarding, habit tracking, journaling, and motivational tools.

## Live Demo

https://visipomo.vercel.app/

## Features

- 50-minute Pomodoro focus timer with break modes
- Customizable YouTube video or image backgrounds
- Background audio support
- Habit Tracker with streak tracking
- To Do Lists, Journals, and Motivational Quotes
- Vision Boards with image uploads
- Session history and user authentication
- Minimizable timer widget
- Frosted glass UI design

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
